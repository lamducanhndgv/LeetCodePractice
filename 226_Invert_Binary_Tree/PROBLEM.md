# Invert Binary Tree

## Mô tả
-   Đảo ngược cây nhị phân.

### Ví dụ

#### Ví dụ 1:
- Input:
```
     4
   /   \
  2     7
 / \   / \
1   3 6   9
```
- Output:
```
     4
   /   \
  7     2
 / \   / \
9   6 3   1
```

#### Ví dụ 2:
- Input:
```
       1
     /   \
    2     3
   / \   / \
  4   5 6   7
 / \
8   9
```

- Output:
```
       1
     /   \
    3     2
   / \   / \
  7   6 5   4
 / \
9   8
```
