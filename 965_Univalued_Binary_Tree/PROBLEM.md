# Univalued Binary Tree

## Description
-   A binary tree is *univalued* if every node in the tree has the same value. Return **true** if and only if the given tree is *univalued*.

### Example
#### Example 1
-   Input
```
       1
      / \
     1   1
    / \   \
   1   1   1
```
-   Output: ```true```

#### Example 2
-   Input
```
       2
      / \
     2   5
    / \   \
   2   2   2
```
-   Output: ```false```
